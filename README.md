## Project Overview

This project is an interactive Sales Dashboard built using Power BI using the Superstore dataset. It helps analyze sales, profit, customer performance, regional trends, and product performance through interactive visualizations.

---

## Business Objectives

- Monitor Total Sales
- Track Total Profit
- Analyze Monthly Sales Trends
- Identify Top Customers
- Identify Top Products
- Compare Regional Performance
- Analyze Category-wise Profit

---

## Dashboard KPIs

- Total Sales
- Total Profit
- Total Orders
- Average Order Value
- Profit Margin

---

## Dashboard Features

- Interactive slicers
- Monthly Sales Trend
- Sales by State Map
- Profit by Category
- Top 10 Products
- Top 10 Customers
- Sales by Region
- Sales by Category

---

## Tools Used

- Power BI
- Power Query
- DAX
- Excel
